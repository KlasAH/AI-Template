# Boilerplate Skill

This is a template for creating new Agent Skills following the agentskills.io specification.

## Usage
When invoked, this skill runs the associated `scripts/example.js` script to process data.

## Requirements
- Node.js environment
- Workspace access

## Example
1. The agent identifies a task that matches this skill's description.
2. The agent executes the underlying script or follows the steps defined here.
