# Boilerplate Skill

Template for creating robust, action-oriented agent skills following the AgentSkills spec.

## Progressive Setup Checklist
- [ ] Read relevant files in `references/` or `assets/` (if any).
- [ ] Determine the context for the agent action.
- [ ] Execute `node scripts/example.js` to begin the process.
- [ ] Verify output matches `assets/template-output.json`.

## Examples
- When a user requests to build a new agentic integration, copy this boilerplate skill as the baseline architecture.

## Guidelines
- Do not provide a menu of options to users; rely on defaults defined in the skill logic.
- Ensure all custom scripts are housed in `scripts/`.
